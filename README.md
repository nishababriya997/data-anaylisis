# Task 5 - Data Analysis on CSV Files

## Objective
Analyze sales data using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Dataset
The dataset contains:
- Name
- Department
- Sales

## Steps Performed
1. Imported required libraries
2. Loaded the CSV file using pandas
3. Checked data using head() and shape
4. Calculated total and average sales
5. Used groupby() to analyze Total sales by Department
6. Created bar chart visualization

## Key Insights
IT department generated the highest total sales.
HR department showed moderate performance.
Finance department contributed steadily to overall revenue.
Sales data was successfully grouped and summarized using groupby() and sum().
Basic data analysis was completed effectively using Pandas.

## Outcome
This project demonstrates basic data analysis skills using Pandas and data visualization using Matplotlib.
